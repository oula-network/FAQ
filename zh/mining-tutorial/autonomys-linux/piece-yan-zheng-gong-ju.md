### Autonomys Piece 驗證工具

可以驗證已生成的 `piece` 數據，運行以下指令即可驗證：

{% code overflow="wrap" %}
```bash
./autonomys-farmer util verify-piece --nats-server nats://192.168.1.1:4222 --nats-server nats://192.168.1.2:4222 --nats-server nats://192.168.1.3:4222
```
